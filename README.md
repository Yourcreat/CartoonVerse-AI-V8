# 🎬 CartoonVerse AI V8

Professional Telegram AI Bot for creating complete YouTube Cartoon Projects using Google Gemini AI.

---

## ✨ Features

- 📖 AI Story Generator
- 🎥 AI Movie Script Generator
- 🖼️ Image Prompt Generator
- 🎬 Video Prompt Generator
- 🎭 Character Memory
- 📦 Project Memory
- 🏷️ YouTube Title Generator
- 🖼️ Thumbnail Prompt Generator
- 🚀 Railway Ready
- 🤖 Telegram Bot


## ⚙️ Environment Variables

Create a `.env` file:

```env
TELEGRAM_BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
PORT=3000
```
